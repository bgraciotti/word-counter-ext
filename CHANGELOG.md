# Changelog

All notable changes to the Word Counter extension will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-01-26

### Added
- Initial release of Word Counter extension
- Real-time text statistics in search bar
- Character count (with spaces)
- Character count (without spaces)
- Word count
- Line count
- Sentence count (approximate)
- Paragraph count
- Individual metric copy action
- Copy all statistics action (Ctrl+C)
- Clear text action (Ctrl+X)
- Color-coded icons for each metric
- Formatted numbers in Brazilian Portuguese locale
- Empty view with instructions
- List-based interface using native Raycast search bar
